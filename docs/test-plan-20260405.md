## Affected Pages/Routes

- `/login` — JWT login page; all subsequent routes depend on auth
- `/patients` — Patient list with search; primary navigation entry point
- `/patients/:id` — Patient detail with study list + DICOM upload dropzone
- `/viewer?studyId=X` — DICOM viewer with CT + RT Structure + RT Dose overlays

## Key Interactions to Verify

### Auth
- Valid credentials → HTTP 200 + JWT cookie set → redirect to `/patients`
- Invalid credentials → HTTP 401 + error message "Invalid credentials"
- Expired JWT → API returns 401 → frontend redirects to `/login`
- Logout → cookie cleared → redirect to `/login`

### Patient Management
- Create patient → appears in list immediately
- Search by name → filters list in <300ms
- Delete patient → study count shown (N+1 fix verified)
- Patient with 0 studies → shows "0" not blank

### DICOM Upload
- Valid DICOM folder drop → progress bar → files appear in study
- Invalid file (non-DICOM) → HTTP 422 → error toast "Invalid DICOM file"
- File > 500MB → HTTP 413 → error toast "File too large"
- Duplicate SOPInstanceUID → silently skipped, returns `{ skipped: true }` (Decision: silent skip)
- Disk full → HTTP 507 → error toast "Storage error"

### DICOM Viewer
- CT series loads → first slice visible in viewport
- Stack scroll (mouse wheel) → slices navigate correctly
- WW/WC tool → window/level adjusts, overlay text updates
- Zoom → viewport zooms, scale text updates
- Layout 1x1 → 2x2 → 4x4 → correct number of viewports

### RT Structure Overlay
- RT Structure file found by modality (not filename) → ROIs appear in StructurePanel
- Toggle ROI checkbox → overlay hides/shows on viewport
- Empty RT Structure (0 ROIs) → no crash, empty structure list displayed
- Slice change → active slice contour updates

### RT Dose Overlay
- RT Dose file found → heatmap appears on CT
- Dose opacity slider → heatmap fades in/out
- Dose color scale → amber diverging scale applied (Decision: amber accent per DESIGN.md)

### Contouring (Brush Tool)
- Select Brush tool → cursor changes
- Draw on CT slice → labelmap updated, visible immediately
- Switch to different ROI → brush paints into new labelmap
- Undo → reverts last brush stroke
- Redo → restores undone stroke

### AI Auto-Contour
- Click "Auto Contour" button → loading spinner
- Phase 1: 501 response → toast "AI service not configured"
- Phase 1: 60-second timeout → toast "AI service timed out" (Decision: timeout confirmed)

## Edge Cases

- Empty patient list → "No patients found" empty state (not blank screen)
- Study with no DICOM files → "No files uploaded" empty state
- RT Structure with 0 ROIs → graceful empty, no console error
- RT Dose with zero-dose pixels → renders as black/near-black, no crash
- Rapid slice scrolling → no memory leak (lazy loading verified)
- Multiple rapid uploads → all complete without race conditions

## Critical Paths

1. **Full end-to-end:** Register → Login → Create Patient → Upload DICOM folder → Open in Viewer → CT images display
2. **RT Structure workflow:** Open study with RTSTRUCT → ROIs appear → toggle visibility → brush edit one ROI → undo
3. **RT Dose workflow:** Open study with RTDOSE → dose heatmap appears → adjust opacity → toggle off/on
4. **Auth security:** Access `/patients` without JWT → 401 → redirect to login (no data exposed)
